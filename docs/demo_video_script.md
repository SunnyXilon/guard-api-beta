# Two-Minute Demo Video Script

## Goal

Show that Guard API helps communities, marketplaces, and creator apps moderate user-generated content quickly.

## 0:00-0:15 Opening

"Guard API is real-time trust and safety infrastructure for products with live user-generated content: communities, marketplaces, and creator apps."

Show the public page and pricing cards.

## 0:15-0:35 Onboarding

Sign up, create a workspace, and show the one-time moderation key.

Say:

"A new customer creates a workspace, gets a server-side moderation key, and immediately sees the first API call."

## 0:35-1:00 First API Call

Copy the onboarding curl command and run it.

Show the response action: `allow`, `review`, or `block`.

## 1:00-1:25 Dashboard

Open Marketplace and Review cases.

Say:

"Every request is saved with model metadata, policy reasons, usage, and a review case when human follow-up is needed."

## 1:25-1:45 Billing

Open Billing and show moderation credits, plans, checkout buttons, and payment management.

Say:

"Customers can start on a trial, upgrade by moderation credits, and manage payment through billing."

## 1:45-2:00 Close

Show Integration center and API docs.

Say:

"Developers can integrate with Node, Python, or any backend. Operators get the review console. The first paid wedge is small UGC teams that need moderation without building a trust-and-safety stack."
