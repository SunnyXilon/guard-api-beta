# Sales Demo Script

## Opening

Guard API helps products moderate user-generated content before spam, scams, abuse, or safety risk reaches users.

## Demo Flow

1. Show the live moderation console.
2. Run a safe message and explain `allow`.
3. Run a scam or harassment sample and explain `review` or `block`.
4. Open the dashboard preview.
5. Show usage, recent decisions, review cases, policy thresholds, and API keys.
6. Show the integration snippets.
7. Explain pricing by request volume and review workflow needs.

## Discovery Questions

- What kind of content do users submit?
- Which harms are most expensive for your team today?
- Do you need auto-blocking, review queues, or both?
- How many moderation events do you expect per month?
- Who reviews edge cases today?

## Close

Offer a private beta with a fixed quota, weekly feedback call, and direct integration help.
